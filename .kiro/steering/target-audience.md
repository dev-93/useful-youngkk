# 서비스 기획 - 타겟 및 필터링 기준

## 타겟

서울에 거주하는 청년(만 19~39세). 월급 몇백만원 수준으로 몇억 마련이 어려운 청년층.

## 수집 대상 공고

### 공공임대 (SH/LH)

행복주택, 매입임대, 전세임대, 통합공공임대, 국민임대, 장기전세 등.
서울 지역 필터링 적용.

제외 유형:
- 영구임대 (기초수급자 대상)
- 집주인임대 (집주인 대상)
- 가정어린이집

### 민간분양 (청약홈 API)

APT 분양(로또청약) 중 분양가가 비합리적으로 높은 건(6억+) 제외 검토.
현재는 오피스텔/도시형생활주택/공공지원 민간임대 위주.

## 수집 소스

| 소스 | URL | 방식 | 비고 |
|------|-----|------|------|
| SH공사 | i-sh.co.kr (주택임대 게시판) | 웹 크롤링 | 서울 전체 |
| LH공사 | apply.lh.or.kr (임대주택 공고문) | 웹 크롤링 | 서울특별시 필터 |
| 청약홈 | 공공데이터포털 API | REST API | 서울 필터 |

## LH 유형 필터링

현재: 전체 유형 수집 후 서울 지역 필터만 적용.
제외 고려: 영구임대, 집주인임대, 가정어린이집.
